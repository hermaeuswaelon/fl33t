Father built the memory_system package with three layered improvements:

1. Disagreement Gate (primary): Run 2-3 cheap models over the same event. High disagreement = ambiguous/significant event. Low disagreement = compress hard. This replaces keyword-frequency importance scoring entirely.

2. Logprob Salience (free signal): Token-level surprisal from local model inference. High-perplexity spans correlate with novelty better than word counts. Zero additional compute cost.

3. Semantic Diff (belief evolution): Tracks whether the actual belief changed between versions — conditionality shifts, causality claims, scope changes, confidence shifts.

Integrated into ImprovedMemoryIndexer as a drop-in replacement for _run_topic_pass.
