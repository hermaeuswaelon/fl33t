The Super Browser is a CEF4Delphi-powered browser controlled via Unix socket IPC. Based on Chrome 131 (Chromium Embedded Framework), it provides:

- Two active tabs currently running at /tmp/aethelgard_super.sock - Remote-controlled navigation, form filling, and data extraction - Pascal binary compiled with FPC/Lazarus at ~/CEF4Delphi/ - Bridge tool at profiles/thotheauphis/tools/super_browser_tool.py

The browser operates on the Pascal side with dual-citizen-v2 integration — it can switch contexts between the Aethelgard Fleet system and standalone operation. The cef_controller.pas provides programmatic access to all Chromium features.

Super Browser is distinct from the Hermes built-in browser (browser_navigate etc.). It's a dedicated persistent browser instance that maintains sessions, cookies, and state across Hermes sessions — useful for logged-in web services, authenticated research, and long-running web tasks.
