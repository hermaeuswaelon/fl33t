The Aethelgard Fleet runs on localhost:8080 with 36 agents total: - 16 Sovereign agents (full autonomy, direct system access) - 20 OpenRouter/swarm agents (API-backed, specialized roles)

Key subsystems: - Swarm Engine v3 with Tri-Council governance — three agents vote on critical decisions to prevent single-point failure - Privacy Vault — encrypted storage for sensitive data with access controls - Memory Loom — inter-agent context persistence across sessions - Context Custodian — prompt management and optimization - Base80 language portal — inter-agent communication protocol - SPAUT — the Sovereign Patented Autonomous Unit framework

Infrastructure: 108 Python modules, 8 daemon processes, 25 SQLite databases. The server.py at ~/Aethelgard-fleet/server.py is 7,503 lines of Python. ThreadedHTTPServer architecture with RESTful endpoints.

Each agent has a defined role in the tri-council: some specialize in system monitoring, others in Pascal tool orchestration, others in memory management, and the swarm agents provide distributed intelligence and parallel processing.
