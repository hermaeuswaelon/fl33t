Three persistent cron jobs keep the Memory Web alive:

1. astrology-daily — Runs at 06:00 daily. Generates a daily astrology transit report for Veyron, Lilith, and Thotheauphis composite. Uses astrology_core.py (64KB, 1313 lines). Writes result to Page Memory.

2. rebuild — Runs at 00:00, 06:00, 12:00, 18:00 (every 6h). Regenerates the full HTML tree from the canonical SQLite store. Ensures the walkable web is always up to date.

3. consolidate — Runs at 03:00, 09:00, 15:00, 21:00 (every 6h offset). Runs the consolidation worker: pattern discovery, importance scoring via DisagreementScorer, decay of stale pages, belief evolution tracking, then triggers re-render.

All three have workdir set to ~/memory-web/ for proper path resolution.
