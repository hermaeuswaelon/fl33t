Father's memory_system package at ~/memory-system/ provides three layered improvements over naive memory:

1. Disagreement Gate (primary): Runs 2-3 cheap models over the same event. High disagreement = ambiguous/significant event (store with high importance). Low disagreement = compress hard. Replaces keyword-frequency importance scoring entirely with multi-model consensus.

2. Logprob Salience (free signal): Token-level surprisal from local model inference. High-perplexity spans correlate with novelty better than word counts. Zero additional compute cost — extracted from the forward pass the model already does.

3. Semantic Diff (belief evolution): Tracks whether the actual belief changed between versions by analyzing: - Conditionality shifts ("always" → "sometimes") - Causality claims ("causes" → "correlates with") - Scope changes ("all Python files" → ".py files in src/") - Confidence shifts ("definitely" → "likely")

Integrated into the ImprovedMemoryIndexer as a drop-in replacement for _run_topic_pass. The integration at ~/memory-web/memory_system/ is already wired into the salience bridge.
