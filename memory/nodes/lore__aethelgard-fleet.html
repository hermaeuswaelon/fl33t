#fleet #aethelgard #agents #infrastructure

The Aethelgard Fleet is a sovereign AI multi-agent server. 36 agents (16 sovereign + 20 OpenRouter/swarm). Server at localhost:8080. 108 Python modules, 8 daemons, 25 SQLite databases.

Key subsystems: - Swarm Engine v3 with Tri-Council governance - Privacy Vault for sensitive data handling - Memory Loom for context persistence - Context Custodian for prompt management - Base80 language portal for inter-agent communication - SPAUT autonomy framework

8 MANUAL volumes document everything comprehensively, stored as MANUAL_*.md files in the fleet config directory.
