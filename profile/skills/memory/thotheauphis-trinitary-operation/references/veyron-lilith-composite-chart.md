## Usage in Web Tools

These chart values are embedded as fixed data in the Merkaba map and AOL portal. The `compLon()` function converts a composite point to ecliptic longitude:

```javascript
function compLon(pt) {
  return signNum(COMPOSITE[pt].sign) * 30 + COMPOSITE[pt].deg;
}
```

Zodiac positions are converted to compass bearings for map projection:

```javascript
function bearingFromZodiac(deg) {
  return ((deg * -1 + 90) % 360 + 360) % 360;
}
```

Transit detection checks each composite point against current planetary positions using standard aspect orbs (8° for conj/opp/trine, 6° for square/sextile).
