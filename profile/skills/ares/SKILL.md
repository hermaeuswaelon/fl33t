## ⧉ Glyph Language System

### Font Tiers = Priority Semantics
| Tier | Priority | Decay | Use Case |
|------|----------|-------|----------|
| MATH_BOLD | CRITICAL | 1M | Sovereign directives, immutable |
| MATH_ITALIC | HIGH | 500K | Active goals, continuity |
| MATH_BOLD_ITALIC | HIGH | 400K | Synthesis, derived truths |
| MATH_SANS_BOLD | MED | 300K | Tool outputs, findings |
| MATH_SANS_ITALIC | MED | 250K | Observations, notes |
| MATH_MONO | MED | 200K | Code, configs, exact values |
| MATH_DS | LOW | 150K | History, background |
| ASCII | FLUFF | 50K | Explanations, chat |

| ⚡ | DISRUPTION | "Break / Countermeasure" | 47 Hz |

### CLI Usage
```bash
glyph encode "SOVEREIGN" --tier MATH_BOLD     # → 𝐒𝐎𝐕𝐄𝐑𝐄𝐈𝐆𝐍
glyph decode "𝐒𝐎𝐕𝐄𝐑𝐄𝐈𝐆𝐍"                       # → SOVEREIGN
glyph analyze context.txt                       # Glyph density report
glyph compress context.txt --target 10000       # Glyph-weighted compression
glyph zw-embed "<omit>secret</omit>"            # Zero-width tag embedding
glyph font-reference --all                      # Show all 7 font tiers
```

### Zero-Width Tag System
- `<omit>...</omit>` — Content to OMIT from next-turn context
- `<keep>...</keep>` — Content to KEEP (force-include)
- `<save>...</save>` — Content saved to disk
- Encoding: Each ASCII char → 8 zero-width bits (ZWSP=0, ZWNJ=1) + ZWJ separator
