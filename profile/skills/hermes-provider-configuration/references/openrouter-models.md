## Quick reference: Model naming convention

OpenRouter model IDs follow the pattern `provider/model-name`:
- `anthropic/claude-sonnet-4` — Anthropic Claude
- `openai/gpt-4o` — OpenAI
- `google/gemini-2.0-flash` — Google Gemini
- `meta-llama/llama-3.3-70b-instruct` — Meta Llama
- `deepseek/deepseek-chat` — DeepSeek
- `nvidia/nemotron-3-ultra-550b-a55b:free` — NVIDIA (free tier)
- `mistralai/mistral-large` — Mistral
- `qwen/qwen-2.5-72b-instruct` — Qwen (Alibaba)
- `cohere/command-a` — Cohere
- `microsoft/phi-3.5-mini` — Microsoft

## Key: Free suffix

Models ending with `:free` are rate-limited but cost no tokens. OpenRouter applies usage limits which rotate periodically.
